# Start MeetingBro

**Start the backend first, then the frontend.**

## Backend

```bash
conda activate MeetingBro
cd app/backend
meetingbro-backend
```

## Frontend (new terminal)

```bash
cd app/frontend
npm run dev
```

The Electron window will open automatically.

# 启动 MeetingBro

**先开后端，再开前端。**

## 后端

```bash
conda activate MeetingBro
cd app/backend
meetingbro-backend
```

## 前端（新终端）

```bash
cd app/frontend
npm run dev
```

Electron 窗口会自动弹出。
